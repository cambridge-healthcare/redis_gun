# RedisGun

For when you need many unique redis-server instances.

## Installation

Add this line to your application's Gemfile:

    gem 'redis_gun'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install redis_gun

## Usage

See the **examples** directory.
